## Simple API Ecommerce with basics features:
- JWT & Refresh Tokens (Authentication System)
- MongoDB & Redis
- Stripe
- Products CRUD
- Cart 
- Coupons System

## TypeScript Configuration:
- npm i -D typescript @types/node
- npx tsc -init
- npm i -D @tsconfig/node23
- npm i -D ts-node

## App config:
- Insert your data to .env file
- run: npm install
- run: npm start
